## 👋 About
End-to-end data analytics project that transforms raw retail sales data into meaningful insights using Python and Power BI.

## 🎯 Objective
- Clean and preprocess messy real-world data  
- Perform exploratory data analysis (EDA) using Python  
- Build interactive Power BI dashboard  
- Generate business insights for decision-making  

## 🧹 Data Cleaning
- Handled missing values and inconsistent formats  
- Fixed data types (Price, numeric fields)  
- Created new features (Discount, Amount)  
- Removed unnecessary columns  

## 📊 Key Insights
- Electronics is the top revenue-generating category  
- Yangon is the highest-performing region  
- Online channel contributes the most revenue  
- Discounts impact overall sales performance
